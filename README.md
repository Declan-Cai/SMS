# SMS
基于Java Web的学生资料管理系统

<br>
## 1.环境

服务器:Tomcat9.0

开发工具:Eclipse、DreamWeaver

数据库:MySql

开发语言：Java+Jsp+HTML5+CSS+Javascript

前端框架：BootStrap4.0

## 2.功能模块：

（1）    管理员登录与退出

（2）    修改管理员密码。

（3）    添加学生信息。

（4）    修改学生信息。

（5）    删除学生信息。

（6）    查询学生信息。
